# Pull

Remote 저장소에서 data를 fetch한 후, 자동으로 merge 하는 명령어이다.

---

#### `$ git pull`

Remote 저장소에서 데이터를 가져온 후, 자동으로 merge 한다.

---

#### `$ git pull -r`
#### `$ git pull --rebase[=false|true|preserve|interactive]`

* `-r` : Rebase하여 pull한다.

---

#### `$ git pull -p`

* `-p` : 실제로 존재하지 않지만, fetch로 전달되는 parameter이다.
