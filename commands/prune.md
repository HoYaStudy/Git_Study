# Prune

Prune all unreachable objects from the object database.

---

#### `$ git prune -n`
#### `$ git prune --dry-run`

-n (--dry-run) option: Do not remove anything; just report what it would remove.

---

#### `$ git prune -v`
#### `$ git prune --verbose`

-v (--verbose) option: Report all removed objects.

---

#### `$ git prune --progress`

--progress option: Show progress.

---
